What Could Possibly Go Wrong?

File doesn't exist

Open, reading or writing to a file or directory? Kind of helps when they actually exist :) Maybe your path is wrong or another process has removed it.

Common Error Codes

  • ENOENT

General

Typically you can check to see if a file exists, before doing operations on it.

POSIX

You can fs.stat a file before operating on it.

Steps to Reproduce

NodeJS

var fs = require('fs');
fs.readFile('doesnotexist', function(err, data) {
  console.log(err);
});
